# Miért jött létre az IPAS?
Az IPAS egy gimnáziumi projektnek indult, mely keretében 7-8 osztályos diákok projekt alapú oktatása lenne. Egy olyan projektet szerettünk volna létre hozni ami az egyetemi portfoliónak is megálná a helyét, de eközben a kisebbeket be tudnánk vezhetni a modern kutatás és a STREAM oktatásba egyaránt.
A projekt megvalósulását a magyar állam is támogatta egy pályázat keretében.
